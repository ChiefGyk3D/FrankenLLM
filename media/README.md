# Media Assets

This directory contains media assets for the FrankenLLM project.

## Required Assets

- `streamelements.png` - StreamElements logo (32x32px recommended)
  - You can download this from your StreamElements branding assets
  - Or use a custom icon representing StreamElements

## Adding the StreamElements Icon

To add the StreamElements icon:

1. Download or create a 32x32px PNG image
2. Save it as `streamelements.png` in this directory
3. The README.md will automatically reference it

Alternatively, you can use any icon service URL in the README if you prefer.
